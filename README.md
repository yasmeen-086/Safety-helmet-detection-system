# Safety-helmet-detection-system
Real-time helmet detection system using computer vision and deep learning

## 🚀 Overview
This project detects whether a person is wearing a helmet in real-time using computer vision techniques.

## 🧠 Problem Statement
Ensuring safety compliance in construction and traffic environments by detecting helmet usage.

## ⚙️ Tech Stack
- Python
- OpenCV
- Deep Learning

## 🔥 Features
- Real-time video detection
- Bounding box predictions
- Reduced flickering across frames
- Works on live webcam feed

## 📊 Results
- Stable predictions across frames
- Improved detection accuracy

## 🚀 Future Improvements
- Add multi-person tracking
- Improve accuracy with advanced models
- Deploy on edge devices
